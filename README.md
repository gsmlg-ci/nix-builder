# nix-builder

[![Build](https://github.com/gsmlg-ci/nix-builder/actions/workflows/build.yml/badge.svg)](https://github.com/gsmlg-ci/nix-builder/actions/workflows/build.yml)

Docker image published to:
- `docker.io/gsmlg/nix-builder`
- `ghcr.io/gsmlg-dev/nix-builder`

## Usage

```bash
# From Docker Hub
docker pull gsmlg/nix-builder:latest

# From GitHub Container Registry
docker pull ghcr.io/gsmlg-dev/nix-builder:latest
```

## Build

```bash
docker build -t gsmlg/nix-builder .
```
